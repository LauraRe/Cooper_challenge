# Cooper-Api

## Prerequisites
* Ruby 2.4.1
* Rails 5.2.0
* PostgreSQL

## Installing

```
bundle install
```
```
rails db:migrate
```
## Running the tests

Run RSpec to perform the tests.
```
$ rspec
```
## Deployment
Deployed at with Heroku.
